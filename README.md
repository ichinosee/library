# library
図書館の司書の方の作業効率をあげるためのサイト

<h1>ねらい</h1>
紙ベースで会員登録や貸し出し管理をしている図書館、図書室などをターゲットに
会員情報や貸し出し情報などが一目でわかるようにして貸し出し台帳などを探す手間を軽減させました。
<br> 
<h1>機能</h1>
・会員登録<br>
・貸し出し/返却<br>
・会員登録と同時に貸し出し対応が同じページで行うことができる<br>
・会員番号から会員情報を検索<br>
・会員情報から貸し出し/返却を行うことができる<br>
